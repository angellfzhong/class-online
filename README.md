## express+ejs+mongodb构建在线学习交流平台

#### 简介
主要包括用户注册，在线聊天，上传/下载资料，发表/查看/删除文章，查看用户详情等功能。
使用express搭建项目框架，定义路由，res.render()渲染页面，mongoDB操作数据库实现数据的增删查改功能，
